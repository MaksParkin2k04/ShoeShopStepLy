# Простая мультитенантная админка

## Проблема
При копировании всего функционала из основного проекта возникло 160+ ошибок компиляции из-за:
- Сложных зависимостей между компонентами
- Отсутствующих пакетов (SkiaSharp, NuGet и др.)
- Неправильных namespace
- Отсутствующих атрибутов и сервисов

## Решение
Создать упрощенную версию с базовым функционалом:

### Что работает:
1. Регистрация компаний
2. Авторизация пользователей
3. Базовая навигация по URL `/{companyName}`
4. Проверка прав доступа

### Что нужно добавить постепенно:
1. Копировать по одной странице админки из основного проекта
2. Исправлять ошибки по мере добавления
3. Добавлять недостающие пакеты по необходимости

### Рекомендация:
Начать с простой рабочей версии и добавлять функционал поэтапно, а не копировать все сразу.

- ✅ Модели Company, CompanyUser
- ✅ Сервис CompanyService  
- ✅ Страницы регистрации и выбора компании
- ❌ Полный функционал админки (требует поэтапного добавления)
