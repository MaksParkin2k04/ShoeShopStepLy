# ShoeShop.MultiTenantAdmin

Мультитенантная система администрирования для управления несколькими компаниями.

## Возможности

- Регистрация новых компаний
- Уникальные URL для каждой компании: `/{shortName}`
- Система авторизации с Identity
- Управление пользователями компаний

## Структура URL

- `/` - Главная страница (перенаправляет на `/companies`)
- `/companies` - Список компаний пользователя
- `/Company/Register` - Регистрация новой компании
- `/{shortName}` - Админка конкретной компании

## Запуск

1. Убедитесь что установлен SQL Server LocalDB
2. Запустите проект: `dotnet run`
3. База данных создастся автоматически
4. Зарегистрируйтесь и создайте компанию

## Модели

- **Company** - Информация о компании
- **ApplicationUser** - Пользователь системы
- **CompanyUser** - Связь пользователя с компанией и его роль

## Технологии

- ASP.NET Core 9.0
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server LocalDB
